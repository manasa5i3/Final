Description: This project aims to develop a Machine Learning-based system that can predict diseases by analyzing symptoms input by users. It helps in early diagnosis and guides patients to consult the right specialist, reducing delays and misdiagnoses.

Objective: To use symptom-based data and predictive models to identify potential illnesses, improving access to primary diagnosis and saving time for both patients and medical professionals.

Key Features:

  • User inputs common symptoms via a form or CLI
  • Machine Learning model trained with disease-symptom mapping
  • Outputs the most probable disease with confidence score
  • Random Forest Classifier used for high accuracy prediction
  • Scalable to include specialist recommendations in future

Technologies Used: Python, Pandas, Scikit-learn, RandomForestClassifier, Flask (optional for web interface)

Description: This project is a console-based application built using Python. It showcases the core principles of Object-Oriented Programming (OOP) such as encapsulation, inheritance, and modularity. The Student Management System is designed to streamline the way student data is handled within a classroom or institution. It provides a user-friendly interface and robust backend logic to manage all student-related operations effectively.

Objective: To create an interactive program that can add, update, delete, and display student details in a secure and structured format using Python's class-based system.

Core Features:

  • Create and store new student records dynamically
  • Update any field in existing records based on student ID
  • Delete student data safely and validate entries
  • Display the list of students with all their academic details
  • Apply inheritance to reuse code and create clean architecture

Technologies Used: Python, OOP concepts, CLI interface

This project is ideal for understanding how to structure and manage code in Python for real-world academic systems. It improves operational efficiency and data accessibility, and serves as a strong base for future web-based or GUI applications in the education domain.
